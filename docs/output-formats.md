# Supported Output Formats

The `rdf-serializer` supports the following output formats:

| Format | Default File Extension | Mime Type | Notes |
| ------ | -------------- | --------- | ----- |
| RDF/XML | .rdf | | |
| OWL/XML | .owl | | |
| Turtle | .ttl | | | 
| [JSON-LD](http://www.w3.org/TR/json-ld/) | .jsonld | | |
| N-Triples | .nt | | |
| N3 | .n3 | | |
| TriG | .trig | | |
